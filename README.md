## octocatcrawler
octocatcrawler is a python script, designed for download all [Octodex](https://octodex.github.com) image.

### System requirements
* Python 2.7.x
* Python 3.x

### Usage
* Get source code from Github:
```
git clone https://github.com/jrtx0/OctodexImgDownloader.git
```
* Run **crawler.py**
``` python
python3 crawler.py
```
